Ayuda

Asistente Agrícola es un software que le brinda recomendaciones de actividades agrícolas diarias para mantener sus cultivos de flores en óptimo estado.


Para guardar datos de una medición

1-Seleccione la fecha puede ser la fecha actual o una anterior

2-Ingrese la cantidad de botones

3-Ingrese los datos de botones en cm

4-Ingrese la medicion de Ph, Densidad y Conductividad electrica

Notas:

* Las medidas de cada uno de los botones no se guardan

* Si deja un campo vacio no se guardará ningún valor


Reportes

* Para generar un reporte PDF de Recomendaciones obtenidas dirijase al menu Historial de Recomendaciones

* Para el reporte EXCEL puede filtrar por rango de fechas

* Para generar un reporte EXCEL de Mediciones ingresadas dirijase al menu Historial de Recomendaciones

* Para el reporte EXCEL puede escoger 4 filtros: Bloque, Cama, N° de muestreo, Variedad y por rango de fechas


Recomendación

Aqui puede estar o no de acuerdo con la recomendación dada por el sistema, si esta de acuerdo de clic en boton si, caso contrario se desplegara una lista de opcion en la cual usted debere escoger cual seria la mejor recomendacion

* Nota: Si no da click en el botón Guardar, la recomendación para la medición se perderá


Usuarios

* Solamente el usuario con rol de ADMINISTRADOR tiene acceso a este opción

* El administrador podrá crear, editar, ver y eliminar usuarios que tengan acceso a la aplicación

* Nota: No se podrá cambiar la contraseña de un usuario desde aquí. Si se le olvidó o perdió la contraseña, deberá recuperarla mediante su correo electrónico
